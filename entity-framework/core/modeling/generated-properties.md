## <a name="value-generation-patterns"></a><span data-ttu-id="a9abf-103">Vzory generování hodnoty</span><span class="sxs-lookup"><span data-stu-id="a9abf-103">Value Generation Patterns</span></span>

### <a name="no-value-generation"></a><span data-ttu-id="a9abf-105">Generování žádná hodnota</span><span class="sxs-lookup"><span data-stu-id="a9abf-105">No value generation</span></span>

<span data-ttu-id="a9abf-113">Pokud Přidání entity do kontextu, který se má hodnota přiřazená k vlastnosti EF se pokusí vložit tuto hodnotu místo generování nové.</span><span class="sxs-lookup"><span data-stu-id="a9abf-113">If you add an entity to the context that has a value assigned to the property, then EF will attempt to insert that value rather than generating a new one.</span></span> <span data-ttu-id="a9abf-114">Vlastnost je považován za mít přiřazenu Pokud není přiřazen výchozí hodnotu CLR hodnotu (`null` pro `string`, `0` pro `int`, `Guid.Empty` pro `Guid`atd.).</span><span class="sxs-lookup"><span data-stu-id="a9abf-114">A property is considered to have a value assigned if it is not assigned the CLR default value (`null` for `string`, `0` for `int`, `Guid.Empty` for `Guid`, etc.).</span></span> <span data-ttu-id="a9abf-115">Další informace najdete v tématu [explicitní hodnoty pro generované vlastnosti](../saving/explicit-values-generated-properties.md).</span><span class="sxs-lookup"><span data-stu-id="a9abf-115">For more information, see [Explicit values for generated properties](../saving/explicit-values-generated-properties.md).</span></span>

> [!WARNING]

>
> <span data-ttu-id="a9abf-118">Například při použití SQL serveru, hodnoty budou automaticky generovány pro `GUID` vlastností (pomocí sekvenčním algoritmem identifikátor GUID systému SQL Server).</span><span class="sxs-lookup"><span data-stu-id="a9abf-118">For example, when using SQL Server, values will be automatically generated for `GUID` properties (using the SQL Server sequential GUID algorithm).</span></span> <span data-ttu-id="a9abf-119">Nicméně pokud určíte, `DateTime` se vygeneruje vlastnost při přidání, pak musíte nastavit způsob, jak hodnoty, které mají být generovány.</span><span class="sxs-lookup"><span data-stu-id="a9abf-119">However, if you specify that a `DateTime` property is generated on add, then you must setup a way for the values to be generated.</span></span> <span data-ttu-id="a9abf-120">Jeden způsob, jak to provést, je k nastavení výchozí hodnoty `GETDATE()`, naleznete v tématu [výchozí hodnoty](relational/default-values.md).</span><span class="sxs-lookup"><span data-stu-id="a9abf-120">One way to do this, is to configure a default value of `GETDATE()`, see [Default Values](relational/default-values.md).</span></span>
