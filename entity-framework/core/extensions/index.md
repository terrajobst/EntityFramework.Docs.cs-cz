# <a name="ef-core-tools--extensions"></a>EF základní nástroje a rozšíření

Nástroje a rozšíření poskytují další funkce pro Entity Framework Core.

Rozšíření jsou vytvořeny pomocí různých zdrojů. Ne všechny rozšíření se udržují v rámci projektu Entity Framework Core. Až budete zvažovat rozšíření třetích stran, ujistěte se, že jste vyhodnocení kvality, licencování, podpora, atd. Chcete-li zajistit, aby že odpovídaly vašim požadavkům.
