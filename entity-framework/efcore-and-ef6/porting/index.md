# <a name="porting-from-ef6-to-ef-core"></a>Přenesení z EF6 do EF Core

Z důvodu zásadních změn v EF Core nedoporučujeme migraci z EF6 na EF Core, pokud pro takovou změnu nemáte dobrý důvod. Podívejte se přesunout z EF6 do EF Core jako port místo upgradu.
